# This is a Hotel Management System Project created by Nishith Upadhyay and Nikunj Hudka.
We have created this C# project using Winforms and GunaUI Toolbox which has new features 
making the interface more attractive.
We have used white and grey colors for the portal and dashboard to show a sense of luxury 
that a hotel can provide.
